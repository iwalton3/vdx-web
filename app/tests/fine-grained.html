Fine-Grained Renderer Tests

About These Tests

These tests validate the fine-grained template renderer before integration with component.js.

  • Static content rendering
  • Dynamic text and attributes
  • Reactive updates (per-binding effects)
  • Event handling
  • x-model two-way binding
  • Refs
  • Control flow (when, each)
  • Effect cleanup
  • Deferred children (for slots)
